Admin Dashboard

You can manage all of the courses offered in your e-learning system from this dashboard. You can create new courses, edit existing courses, and delete courses that are no longer needed.

You can manage all of the users who have registered for your e-learning system from this dashboard. You can view user profiles, reset passwords, and delete users if necessary.

You can manage all of the assessments that are available in your e-learning system from this dashboard. You can create new assessments, edit existing ones, and delete assessments that are no longer needed.
